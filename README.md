# ecommerce-sales-analysis-using-SQL
MySQL project for analyzing e-commerce sales data and generating actionable business insights through SQL queries.

## Project Overview
Analyzed an e-commerce sales dataset using MySQL to derive business insights.

## Skills Used
- SQL
- MySQL
- Aggregate Functions
- GROUP BY
- HAVING
- CASE
- Subqueries
- Window Functions
- CTEs

## Key Analysis
- Total Revenue
- Monthly Sales Trend
- Top Customers
- Revenue by Category
- Revenue by Region
- Payment Method Analysis
- Customer Ratings
- Delivery Performance
- Customer Ranking

## Dataset
5000 E-commerce Sales Records

- MySQL Workbench
